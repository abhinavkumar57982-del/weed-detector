# 🌱 WeedDetect AI – Smart Weed Detection System

An AI-powered weed detection system developed for the **Weed Buster** competition at Shristi Tech Fest, NERIST.  
This project uses **Computer Vision** and **Deep Learning** to identify weeds from crop images and support precision agriculture.

🏆 **Achievement:** Secured **3rd Prize** in the Weed Buster Event.

---

# 🚀 Project Overview

WeedDetect AI is designed to help identify weeds from agricultural crop images using a trained deep learning model.

The model was trained on approximately **2700+ crop and weed images** collected from Kaggle datasets.  
Training and optimization took nearly **10–12 continuous hours** to improve prediction accuracy and real-world performance.

This project demonstrates the use of AI in agriculture for:
- Smart farming
- Automated weed identification
- Crop monitoring
- Precision agriculture solutions
